Prédiction des Propriétés ADMET

Contexte

Ce projet vise à développer un modèle de classification multi-étiquettes supervisée capable de prédire trois propriétés ADMET (Absorption, Distribution, Métabolisme, Excrétion, Toxicité) pour diverses molécules, une étape cruciale dans la découverte de médicaments.

Objectifs

  • Prédire les propriétés ADMET de molécules candidates
  • Identifier rapidement des candidats-médicaments avec des profils ADMET favorables
  • Réduire le coût et le temps des évaluations expérimentales

Technologies utilisées

Python RDKit Scikit-learn Pandas NumPy Streamlit

Résultats

Développement d'un modèle capable de prédire avec précision les propriétés ADMET à partir des structures moléculaires, permettant d'identifier rapidement les candidats les plus prometteurs pour des tests ultérieurs.

Segmentation d'Images Médicales

Contexte

Développement d'une API de segmentation d'images médicales utilisant des techniques de few-shot learning pour faciliter l'apprentissage avec un nombre limité d'images annotées.

Objectifs

  • Créer un outil de segmentation d'images médicales précis
  • Implémenter des techniques de few-shot learning pour réduire le besoin en données annotées
  • Développer une interface utilisateur intuitive pour les professionnels de santé

Technologies utilisées

Python FastAPI PyTorch OpenCV Docker Streamlit

Fonctionnalités

  • Téléversement d'images médicales
  • Prédiction de segmentation en temps réel
  • Interface utilisateur intuitive
  • Documentation automatique avec Swagger UI
